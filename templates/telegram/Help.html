cosmos-transactions-bot v{{ . }} This bot can track any transactions on any Cosmos-compatible network and report them on different reporters (currently, only Telegram is supported). Created by 🐹 Quokka Stake with ❤️. The bot can understand the following commands: - /help - display this message - /status - see nodes status - /config - see app config - /alias [chain] [address] [alias] - set an alias for wallet - /aliases - see wallets aliases
